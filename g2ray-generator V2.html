⚠️ مهم: برای هر اکانت GitHub فقط یک بار از این روش استفاده کنید. اگر چند Codespace با فایل‌های یکسان بسازید، احتمال بن شدن وجود دارد. این Generator همین مشکل را حل می‌کند.
1

ساخت اکانت GitHub

به github.com بروید و یک اکانت جدید بسازید. برای هر پروکسی جداگانه یک اکانت مجزا توصیه می‌شود.

2

Fork کردن ریپو

ریپوی اصلی g2ray را Fork کنید. سپس وارد ریپوی Fork شده خودتان شوید.

3

ساخت فایل‌های یکتا با این ابزار

در تب Generator روی ساخت فایل‌های یکتا بزنید و سپس همه فایل‌ها را دانلود کنید.

نکته: UUID شما هر بار که این HTML را باز کنید کاملاً رندوم است و با هیچ فورک دیگری یکسان نخواهد بود.
4

جایگزین کردن فایل‌ها در ریپو

در ریپوی GitHub خود، پوشه .devcontainer را باز کنید. هر فایل را یکی‌یکی با نسخه دانلود شده جایگزین کنید (آپلود و commit).

ساختار صحیح پوشه .devcontainer:

📁 .devcontainer/
  ├── config.json
  ├── Dockerfile
  ├── devcontainer.json
  ├── setup.sh
  ├── show-link.sh
  └── start.sh
5

ساخت Codespace

در ریپوی GitHub روی Code کلیک کنید، تب Codespaces را انتخاب کنید و New codespace را بزنید. چند دقیقه صبر کنید تا build کامل شود.

6

دریافت لینک اتصال

بعد از build، لینک VLESS به صورت خودکار در ترمینال نمایش داده می‌شود. اگر نشان نداد، این دستور را بزنید:

show-link.sh

لینک را کپی و در کلاینت وارد کنید.

مدیریت Codespace بدون باز کردن GitHub
★

روشن/خاموش از ترمینال

با نصب GitHub CLI می‌توانید از هر جایی Codespace را کنترل کنید:

gh codespace list
gh codespace start -c CODESPACE_NAME
gh codespace stop -c CODESPACE_NAME
gh codespace ssh -c CODESPACE_NAME
اپ موبایل: اپ GitHub Mobile را نصب کنید — از آنجا بدون مرورگر می‌توانید start/stop کنید.
★

سقف رایگان ماهانه

اکانت رایگان GitHub ماهانه 120 core-hour دارد. چون از ۲ هسته استفاده می‌شود، معادل 60 ساعت در ماه است. بعد از استفاده Codespace را stop کنید تا quota هدر نرود.

★

کلاینت‌های سازگار

برای اتصال از این کلاینت‌ها استفاده کنید:

اندروید: v2rayNG — Nekobox
ویندوز: v2rayN — Nekoray
iOS: Streisand — FoXray — Shadowrocket
مک: FoXray — V2Box

پینگ نمی‌دهد — مشکل دارد؟

خیر — Codespace hostname از نوع HTTPS است و ICMP (پینگ) همیشه بلاک است. برای تست اتصال از curl استفاده کنید:

curl -I https://CODESPACE_NAME-443.app.github.dev/

اگر پاسخ 200 یا 502 برگشت، سرور آنلاین است.

?

Codespace خودش خاموش می‌شود

این رفتار طبیعی GitHub است. اسکریپت start.sh یک keep-alive داخلی دارد که هر ۵ دقیقه یک سیگنال می‌فرستد. اگر باز هم خاموش شد، timeout را در تنظیمات GitHub روی ۲۴۰ دقیقه بگذارید.

?

خطای geoip.dat هنگام اجرا

در ترمینال Codespace بزنید:

sudo curl -sL "https://github.com/v2fly/geoip/releases/latest/download/geoip.dat" -o /usr/local/bin/geoip.dat
sudo curl -sL "https://github.com/v2fly/domain-list-community/releases/latest/download/dlc.dat" -o /usr/local/bin/geosite.dat
?

چرا باید فایل‌ها را یکتا کنم؟

اگر همه فورک‌ها UUID و config یکسان داشته باشند، GitHub آن‌ها را به عنوان یک pattern شناسایی می‌کند و اکانت‌ها را بن می‌کند. این Generator هر بار UUID و نام متفاوت تولید می‌کند.
